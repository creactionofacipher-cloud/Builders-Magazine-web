# **Design Guide**

## **Purpose**

This document defines the visual language and design principles for the Builders Magazine website.

The goal is not simply to build a modern website, but to create a premium digital editorial experience inspired by high-quality printed magazines.

Every design decision should reinforce the identity of Builders Magazine.

---

# **Design Philosophy**

Builders Magazine is not:

- a corporate website
- a news portal
- a technology blog
- an online marketplace

It is a curated editorial platform built around photography, storytelling, craftsmanship, and motorcycle culture.

The website should feel timeless rather than trendy.

---

# **Design Keywords**

The visual language should be described by the following words:

- Editorial
- Premium
- Minimal
- Bold
- Authentic
- Industrial
- Mechanical
- Emotional
- Cinematic
- Spacious

Avoid anything that feels generic or overly commercial.

---

# **User Experience**

The experience should resemble reading a premium printed magazine.

Content should never feel compressed.

Large photographs.

Strong typography.

Comfortable reading rhythm.

Generous whitespace.

Minimal interface distractions.

---

# **Visual Hierarchy**

The order of importance should always be:

1. Photography
2. Headlines
3. Story
4. Interface

The interface exists to support the content, never compete with it.

---

# **Layout Principles**

Use a clean grid.

Prefer large visual blocks.

Avoid unnecessary decorations.

Allow photographs to dominate the page.

Layouts should breathe.

Content should never feel crowded.

---

# **Typography**

Typography is one of the strongest visual elements.

Requirements:

- Strong editorial headlines
- Highly readable body text
- Clear hierarchy
- Consistent spacing
- Comfortable line length

The typography should resemble modern print magazines rather than blogs.

---

# **Photography**

Photography is the primary visual language.

Images should be:

- large
- high quality
- immersive
- authentic

Avoid stock photography.

Every image should tell a story.

Photography should receive more visual attention than UI elements.

---

# **Color Palette**

The palette should remain minimal.

Primary colors:

- Black
- White

Secondary colors:

- Neutral gray tones

Accent colors should be used sparingly.

The motorcycle projects themselves should provide most of the visual color.

---

# **Components**

Every component should be simple.

Avoid excessive borders.

Avoid heavy shadows.

Avoid unnecessary animations.

The content should always remain the focus.

---

# **Buttons**

Buttons should feel understated.

Simple.

Readable.

Functional.

Avoid oversized call-to-action elements.

---

# **Cards**

Cards should not resemble commercial product cards.

They should resemble editorial layouts.

Photography first.

Text second.

Interface last.

---

# **Navigation**

Navigation should be minimal.

Easy to understand.

Consistent across the entire website.

Users should reach any important content within two or three clicks.

---

# **Motion**

Animation should support the reading experience.

Preferred animations:

- subtle fade
- smooth transitions
- gentle image reveals

Avoid:

- bouncing
- spinning
- exaggerated motion

Motion should never distract from the content.

---

# **Responsive Design**

The project follows a mobile-first approach.

The experience should remain consistent across:

- mobile
- tablet
- desktop

The desktop version should take advantage of large photography and generous layouts.

---

# **Editorial Feeling**

The website should evoke the feeling of browsing a premium motorcycle magazine.

The reader should slow down, explore, and enjoy the stories rather than rush through the content.

The design should encourage discovery.

---

# **Inspiration**

The visual direction should draw inspiration from:

- premium editorial magazines
- photography books
- exhibition catalogs
- architecture publications
- luxury print design

Avoid inspiration from SaaS dashboards, startup landing pages, or conventional e-commerce websites.

---

# **Design Principles**

Every design decision should satisfy the following principles:

- Content First
- Photography First
- Editorial Layout
- Timeless Design
- Simplicity
- Performance
- Accessibility
- Scalability

Whenever there is uncertainty, choose the solution that makes the content more prominent.

---

# **Future Design Evolution**

The MVP establishes the visual foundation.

Future versions may introduce:

- richer editorial layouts
- immersive galleries
- interactive storytelling
- long-form reading experiences
- advanced transitions

However, the core visual language should remain consistent.

Builders Magazine should always feel like a premium printed publication translated into the digital world.
