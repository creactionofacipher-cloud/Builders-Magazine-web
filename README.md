# Builders Magazine

Digital platform for Builders Magazine — a motorcycle magazine dedicated to custom motorcycle culture.

## Project Goal

The website is a digital extension of the printed Builders Magazine.

Main goals:

- Present Builders Magazine and its philosophy.

- Showcase every magazine issue.

- Publish exclusive digital stories.

- Present Builders Cup and its participants.

- Build and grow the Builders Magazine community.
